# Project
В данном проекте будут проделаны следующие действия:
● Создаём файл README.md в своем репозитории, используя функцию
"Add file" на странице репозитория, и добавьте в него описание вашего
проекта.
● Создаём новую ветку в своем репозитории, используя кнопку "Branch:
master" на странице репозитория.
● Вносим изменения в файл README.md в созданной ветке, используя
кнопку "Edit" на странице файла, и добавьте в него новые строки.
● Создаём pull request на GitHub для слияния ваших изменений в
основную ветку проекта. Опишите изменения в вашем pull request.
● Создаём тег для релиза вашего проекта на GitHub, используя кнопку
"Releases" на странице репозитория, и добавьте описание и версию проекта.
● Проверим, что все изменения успешно отправлены на GitHub и
внесены в основную ветку проекта.
